# Portfolio Backend

This is the backend API for my personal portfolio website. It handles contact form submissions and connects to a MongoDB database.

## Tech Stack
* **Node.js**: Runtime environment
* **Express.js**: Web framework
* **MongoDB**: Database for storing messages
* **Mongoose**: ODM for MongoDB

## Features
* RESTful API for handling messages.
* Database connection with MongoDB Atlas.
* CORS enabled for frontend communication.
